Family Exporter - Help

Description

Family Exporter is a lightweight, efficient tool designed to streamline the batch export of Revit families into organized folders. It's ideal for documentation, library management, or preparing Revit families for external sharing.

Key Features

  • Select and export multiple families at once
  • Maintain category-based folder structure in the output
  • Define custom export directory

Steps to Run the Tool

  1. Open Revit 2021 or later.
  2. Load your Revit project containing the families you want to export.
  3. Go to WBD | Plugins tab in Revit.
  4. Click on Family Exporter button.
  5. In the window that opens:
    • Select families from the list.
    • Choose export directory.
    • Enable/disable category folder structuring.
  6. Click Export.
  7. Wait for confirmation message.
  8. Navigate to your export directory to review results.

Why Use Family Exporter?

  • Free and regularly updated for the latest Revit versions
  • Exports families using a clear, logical structure
  • User-friendly, focused interface for fast operation
  • More stable than many alternatives, especially in recent Revit versions
  • Removes manual sorting or renaming work
